# Data Jobs Dashboard

## Introduction 
This dashboard provides a comprehensive overview and drill-down analysis of the Data Jobs market, offering insights into job trends, salaries, global distribution, and platform prevalence.

## Interactive Visualizations:

- KPI Cards: Clear, prominent display of aggregate metrics.

- Donut/Pie Charts: Effective visualization of True/False percentages (e.g., the 15% WFH for Data Engineers).

- Time Series/Line Chart: Tracking job market trend over 2024.

- Scatter Plot: Analyzing the relationship between Hourly v/s Yearly Salary.

- Map Visual (Bing Maps integration): Displaying Global Job Distribution with density indicated by bubble size.

- Bar Charts: Ranking job titles by count and comparing job platforms.

## Report Interactivity & Navigation:

- Drill-Through Action: Implemented navigation from the high-level Overview Dashboard to the detailed Data Engineer Drill-down page, focusing the report on a specific role.

- Slicers/Filters: The "Select Job Title" filter enables dynamic filtering across the entire report.
